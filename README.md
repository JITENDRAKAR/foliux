# FOLIUX - Investment Tracking System

FOLIUX is a rule-based investment tracking system designed for disciplined wealth creation. It provides automated signals, portfolio optimization, and comprehensive market insights.

## Features
- **Automated Signals**: Rule-based BUY/SELL/HOLD recommendations.
- **Multi-Asset Support**: Stocks, ETFs, Mutual Funds, NPS, and Coins.
- **Family Portfolio**: Consolidated view of linked family accounts.
- **Real-time Market Data**: Live tracking of Indian and Global indices.
- **Strategy Management**: Portfolio rebalancing and strategy sheets.

## Tech Stack
- **Backend**: Django (Python)
- **Database**: MySQL / SQLite
- **Frontend**: Bootstrap 5, Chart.js, FontAwesome
- **Deployment**: IIS / Linux (sh scripts included)

## Deployment
Use `deploy.sh` for Linux deployments or the included `web.config` for IIS.
